# پوشه تصاویر برنامه (App Showcase Images)

عکس‌های معرفی برنامه را در این پوشه قرار دهید.

## نام فایل‌های مورد نیاز برای اسلایدر صفحه اصلی:

| نام فایل | توضیح | اندازه پیشنهادی |
|----------|--------|------------------|
| `app-1.jpg` | داشبورد مدیریت هوشمند | 1280×720 (16:9) |
| `app-2.jpg` | گزارش‌های تحلیلی | 1280×720 (16:9) |
| `app-3.jpg` | اپلیکیشن موبایل | 1280×720 (16:9) |
| `app-4.jpg` | امنیت و احراز هویت | 1280×720 (16:9) |

## نکات:
- فرمت پیشنهادی: JPG یا PNG یا WebP
- نسبت تصویر: 16:9 (مثلاً 1280×720 یا 1920×1080)
- اگر عکسی موجود نباشد، یک طرح جایگزین زیبا به‌طور خودکار نمایش داده می‌شود
- برای تغییر عناوین و توضیحات، فایل `src/components/AppShowcase.tsx` را ویرایش کنید

## افزودن اسلاید جدید:
در فایل `src/components/AppShowcase.tsx`، آرایه‌ی `slides` را ویرایش کنید.
